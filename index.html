¿Qué son las aplicaciones híbridas?
Las aplicaciones híbridas, a diferencia de las nativas, son aquellas capaces de funcionar en distintos sistemas operativos móviles. Entre ellos: “Android”, “iOS” y “Windows Phone”. De esta manera, una misma app puede utilizarse en cualquier smartphone o tablet, indistintamente de su marca o fabricante.
Para ello, estas aplicaciones tienen componentes que permiten la adaptabilidad de un mismo código a los requerimientos de cada sistema.


¿Cuándo usar aplicaciones híbridas?
Cuando se busca que la aplicación que se va a implementar, esté en distintos sistemas operativos o plataformas, pues si se busca esto y se programa de forma nativa para cada plataforma se va a llevar tiempo.

Conclusión
Las aplicaciones híbridas, han dado una alternativa más a los desarrolladores pues la idea es que una aplicación sea lanzada en un número más elevado de plataformas para que esta tenga un mayor alcance. De tal forma que si lo que se busca es tener este alcance lo ideal es usar una aplicación híbrida. Sin embargo si lo que se busca es tener una aplicación más enfocada a un solo sistema operativo, por la razón que sea, es mejor utilizar una aplicación nativa, como es el caso de un gran número de aplicaciones para iOS
